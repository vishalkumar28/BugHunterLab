# BugHunterLab Workflow

## End-to-end lifecycle

1. Scope analysis: classify target complexity and initial attack surface.
2. Reconnaissance: enumerate assets, live hosts, endpoints, and exposed services.
3. Vulnerability intelligence: review bug classes, payload libraries, and attack conditions.
4. Testing methodology: follow endpoint-specific checklists and manual guides.
5. Proof collection: store screenshots, request/response details, and analyst notes.
6. Validation: assign severity, assess impact, and prepare a clean PoC.
7. Reporting: export a structured report in Markdown or plain text.
